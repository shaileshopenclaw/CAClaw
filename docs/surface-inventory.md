# CAClaw Surface Inventory

This page maps the main user-facing surfaces in this repository.

## Main surfaces

### GitHub landing surface
- `README.md`

### Plugin metadata
- `package.json`
- `openclaw.plugin.json`
- `index.ts`

### User docs
- `docs/`

### Visual assets
- `assets/`

### CA workflow content
- `skills/`

## Why this matters

When you update CAClaw, you usually need to check more than one place.

For example:

- a name change affects README, assets, and plugin metadata
- a workflow update affects skills and docs
- an install update affects README and docs together
