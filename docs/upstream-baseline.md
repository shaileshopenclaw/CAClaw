# OpenClaw Relationship Baseline

This page explains the relationship between CAClaw and OpenClaw.

## Current position

CAClaw is an add-on for OpenClaw.

That means:

- OpenClaw is the platform
- CAClaw is the Chartered Accountant workflow layer
- CAClaw should stay as additive as possible

## Practical meaning

When OpenClaw changes, CAClaw should adapt where needed, but it should not try to become a separate competing core product.

## Good default

If a future change can be done inside:

- skills
- docs
- assets
- plugin runtime files

then it should usually stay there instead of pushing deep changes into OpenClaw core.
