# Long Form Audit Report (LFAR) — Comprehensive Guide
**Based on RBI Circular DOS.CO.PPG./SEC.01/11.01.005/2020-21 dated 05.09.2020 | ICAI GN 2025**

---

## What is LFAR?

LFAR (Long Form Audit Report) is a **management report** — NOT an audit report — submitted by:
- **Branch Auditors** (SBA) → to Bank Management / Statutory Central Auditor
- **Statutory Central Auditors** (SCA) → to the Audit Committee of the Board

Originally introduced by RBI in 1985, revised in 1993, 2003, and **comprehensively revised in September 2020** (effective FY 2020-21 onwards).

**Key Distinction:**
```
Audit Report = Opinion on Financial Statements (Public Document)
LFAR = Detailed management report on specific areas (Confidential, internal)

LFAR is in ADDITION to the statutory audit report.
All significant findings in LFAR impacting financials must be 
also reflected in the Audit Report / MOC.
```

**Submission Deadline:** 30th June every year.

## LFAR Structure (Branch Level — Annex II of RBI Circular)

### SECTION I: ADVANCES

#### I-1. General

```
a. Whether the branch has complied with the sanction terms 
   and conditions stipulated by the sanctioning authority?
   → Check all loan files: rate of interest, collateral, end-use
   → Flag deviations in LFAR with account name and amount

b. Is the credit monitoring and supervision satisfactory?
   → Is there a follow-up system for overdue accounts?
   → Are periodic visits to borrower premises made and documented?

c. Does the branch maintain adequate records of loan documents?
   → Document checklist vs actual documents in file
   → Flag missing documents (title deed, insurance, stock statements)
```

#### I-2. Documentation

```
For EACH sample account, check and record:
[ ] Sanction letter — signed by borrower?
[ ] Security documents (mortgage deed, hypothecation, pledge)
[ ] Personal guarantee of directors/proprietor
[ ] Board resolution (for corporate borrowers)
[ ] Insurance policy (valid, bank as mortgagee/lossee)
[ ] Stock statements (monthly for CC/OD) — any gaps?
[ ] Stock audit report (for limits >₹5 crore, or per bank policy)
[ ] Legal title search (within last 3 years for mortgaged property)
[ ] Property valuation (within last 3 years)
[ ] Audited financials of borrower (latest available)
[ ] IT assessment orders / GST returns (for income verification)
[ ] ROC search (for corporate borrowers)
[ ] CIBIL check at sanction — adverse features noted?

Common LFAR Documentation Findings to Report:
→ "Title deed not available on record for Account No. XYZ, O/s ₹___ lakhs"
→ "Insurance policy expired on dd/mm/yyyy for Account No. XYZ"
→ "Stock statements not submitted since mm/yyyy for Account No. XYZ"
→ "Valuation report more than 3 years old for Account No. XYZ"
```

#### I-3. Monitoring and Supervision

```
[ ] Periodic credit review conducted? (Annual for most accounts)
[ ] Branch visit report in file for CC/OD accounts?
[ ] Account under Watch/SMA — enhanced monitoring?
[ ] Concurrent audit observations on advances — complied?
[ ] Any account where DP reduced significantly but outstanding not reduced?
[ ] Any account restructured without proper RBI approval?
```

#### I-4. Investments (Branch Level)

```
[ ] Branch investments held (if applicable)
[ ] Verification of investment certificates / SGL balance
[ ] Valuation: HTM/AFS/HFT as per RBI norms
[ ] Any overdue/matured investments not encashed?
[ ] NPI (Non-Performing Investment) identified and provisioned?
[ ] Physical verification of investments on 31 March
```

#### I-5. NPA — Core LFAR Section

```
(a) Overall NPA position:
    → Gross NPA amount and number of accounts
    → Movement during year (additions, reductions, upgrades, write-offs)
    → Category-wise: Sub-standard, D1, D2, D3, Loss

(b) Accounts where auditor DISAGREES with bank classification:
    Format:
    | Account No. | Name | O/s Balance | Bank Class | Auditor's Class | Reason |
    
(c) SMA Monitoring:
    → Is branch tracking SMA-0, SMA-1, SMA-2 accounts?
    → Reporting to CRILC for accounts ≥₹5 crore?
    → Action plan for SMA accounts?

(d) Large NPA accounts (>₹5 crore):
    → List with name, amount, date of NPA, action taken
    → SARFAESI notice served? 60-day notice expired?
    → Recovery progress — OTS/compromise/legal/IBC

(e) Provisions:
    → Are provisions adequate as per RBI IRACP norms?
    → Any provision shortfall — amount and accounts
    → PCR (Provision Coverage Ratio) computation

(f) Restructured Accounts:
    → List of restructured accounts, monitoring status
    → Is monitoring period being tracked?
    → Accounts that slipped after restructuring?

(g) Recovery:
    → Amount recovered from NPA accounts during year
    → Recovery from technically written-off accounts
    → SARFAESI proceedings status — sales conducted?
    → OTS/compromise settlements — RBI approval (if required)?

(h) Written-off accounts:
    → Technical write-offs during year
    → Full write-offs during year
    → Recovery from written-off accounts this year

(i) URFSL Accounts:
    → Un-resolved Financial Stressed Loans under RBI 2019 Framework
    → Additional provisioning status

(j) Special Accounts:
    → Export bills overdue — FEMA compliance
    → Bills purchased/discounted — maturity profile
    → Devolved LCs / invoked BGs
```

#### I-6. Fraud and Irregularities in Advances

```
[ ] Any accounts with suspected fraud — reported to HO/RBI?
[ ] Inflated stock statements — independent stock audit findings
[ ] Diversion of funds — end-use verification done?
[ ] Multiple financing of same asset — verification done?
[ ] Any circular transactions through current accounts identified?
[ ] Connections between borrowers — related party lending?
[ ] Red flags in operations not reported by concurrent auditors?

LFAR Reporting Format for Fraud:
"Account No. ___, Name: ___, O/s: ₹___ Lakhs — Suspected fraud on 
account of [describe]. FMR filed on [date] / Yet to be filed [reason]."
```

### SECTION II: LIABILITIES

#### II-1. Deposits

```
(a) Interest Calculation:
    [ ] Verify correct interest rates applied — check system master vs rate card
    [ ] EBLR linkage for all floating rate deposits (if any)
    [ ] Senior citizen differential interest: correctly applied?
    [ ] Bulk deposit rates: Board-approved? Deviations noted?
    [ ] Interest compounding frequency: quarterly for SB, per FD deed for FDs

(b) Dormant / Inoperative Accounts:
    [ ] SB/CA dormant for 2+ years: Is KYC refresh mandated?
    [ ] Operations restricted as per RBI guidelines?
    [ ] No transactions reversed without proper authorisation?

(c) DEAF (Depositor Education and Awareness Fund):
    [ ] Unclaimed deposits (10+ years): transferred to DEAF in RBI?
    [ ] Certificate of transfer obtained?
    [ ] Claims process in place for depositors?

(d) TDS Compliance:
    [ ] TDS deducted at 10% on interest >₹40,000 (₹50,000 for senior citizens)?
    [ ] Form 15G/15H obtained where applicable?
    [ ] TDS deposit to government account within 7 days of deduction?
    [ ] Form 16A issued within prescribed time?
    [ ] TDS return (Form 26Q) filed quarterly on time?
```

#### II-2. Borrowings

```
[ ] Inter-bank borrowings — properly authorised and accounted?
[ ] Sub-ordinated debt / bonds — interest paid as per terms?
[ ] Refinance (NABARD/NHB/SIDBI/MUDRA): end-use confirmed?
[ ] RBI liquidity facilities (LAF/MSF): within permissible limits?
```

---

### SECTION III: PROFIT & LOSS ACCOUNT

```
(a) Income Recognition:
    [ ] All income from NPA accounts reversed?
    [ ] Commission income — on completion of service, not upfront?
    [ ] Processing fee income: amortised over loan tenure (Ind AS)?
    [ ] No income recognised on accounts where receipt doubtful?

(b) Expenditure:
    [ ] All expenses properly authorised and supported by vouchers?
    [ ] Provisions adequate (NPA, standard assets, depreciation)?
    [ ] Outstanding expenses accrued (rent, utilities, audit fees)?
    [ ] No fictitious expenditure entries?
    [ ] All prepaid expenses correctly stated?

(c) Reconciliation of P&L:
    [ ] Opening balance of P&L agrees with prior year closing?
    [ ] P&L balance matches CBS P&L account?
    [ ] Any exceptional items disclosed separately?
```

---

### SECTION IV: GENERAL

#### IV-1. Gold / Bullion / Security Items

```
[ ] Gold held by branch — weight and purity verified?
[ ] Gold loan portfolio — LTV ratios maintained per RBI guidelines?
    (Max LTV for Gold Loans: 75% — RBI circular)
[ ] Gold auction conducted where mandated?
[ ] Locker inventory — physical verification conducted?
[ ] Items in safe custody — register maintained?
[ ] Stamps, travellers' cheques — counted and recorded?
```

#### IV-2. Books and Records

```
[ ] CBS data backed up regularly?
[ ] Books of account maintained as per Banking Regulation Act?
[ ] Vouchers preserved (minimum 5 years)?
[ ] Account statements generated from CBS — verified for completeness?
[ ] No manual entries outside CBS without proper approval?
[ ] Deleted/reversed entries authorised?
```

#### IV-3. Inspection / Concurrent Audit

```
[ ] Concurrent Audit Reports for all months of FY — obtained?
[ ] Compliance of prior concurrent audit observations — status?
[ ] RBI Inspection (Section 35) observations — complied?
[ ] Internal Audit observations — outstanding items?
[ ] Prior year LFAR adverse remarks — compliance obtained?
```

#### IV-4. Frauds

```
[ ] List of frauds detected at branch during FY
[ ] Amount, nature, date of detection, whether reported to RBI
[ ] FMR (Fraud Monitoring Return) filed timely?
[ ] For frauds ≥₹1 crore: filed to RBI within 3 weeks of detection
[ ] Recovery proceedings initiated?
[ ] Internal inquiry / CBI report (if applicable)?
[ ] Provision made for fraud amounts not recovered?
```

#### IV-5. Compliance with RBI Guidelines

```
[ ] KYC / AML: all accounts KYC-compliant?
[ ] Interest rates: EBLR/MCLR as applicable
[ ] Priority Sector: branch's contribution to overall PSL target
[ ] CDR/SDR: compliance if applicable
[ ] Basel III: contribution data provided to SCA correctly
[ ] IT security: incidents reported to CISO/RBI?
```

#### IV-6. MIS and Reporting

```
[ ] MIS reports submitted to HO on time?
[ ] False/incorrect data submitted to HO? Any detected discrepancies?
[ ] BSR (Basic Statistical Return) data accurate?
[ ] NPA data in RBI returns matches CBS?
```

## Special LFAR Sections

### A. For Forex Branches

```
[ ] NOSTRO accounts: reconciled, stale items cleared
[ ] Forward contracts: marked to market, limits observed
[ ] PCFC/EPC accounts: interest at RBI-prescribed rates
[ ] EEFC/RFC accounts: compliance with FEMA rules
[ ] Export bills outstanding >6 months: regularised or reported?
[ ] Import LC outstanding after due date: payments regularised?
[ ] Dealing room: limits observed, front/back office segregation?
[ ] FEDAI guidelines compliance
[ ] SWIFT controls compliance
```

### B. For Branches with Large Advances (>₹25 crore exposure)

```
[ ] TEV (Techno Economic Viability) studies done for project finance?
[ ] Escrow mechanism in place for project loans?
[ ] DSRA (Debt Service Reserve Account) maintained?
[ ] Project completion certificates received?
[ ] Consortium meetings attendance records?
[ ] NPA accounts >₹1 crore: detailed account-wise comments
[ ] Top 10 NPA accounts: in-depth analysis
```

### C. For Recovery / NPA Branches

```
[ ] SARFAESI proceedings: 60-day notice, possession, auction progress
[ ] DRT proceedings: status, expected recovery
[ ] Compromise/OTS settlements: Board/RBI approval obtained?
[ ] NARCL/NARCL referrals: documentation
[ ] Write-offs during year: RBI-required preconditions met?
[ ] IBC proceedings: NCLT admissions, IRP/RP appointments
```

## Pro Tips for Filling LFAR

```
TIMING:
→ Start filling LFAR from Day 1 of audit, section by section
→ Assign LFAR sections to team members alongside fieldwork
→ Do NOT leave LFAR to the last day

REPORTING STYLE:
→ Be specific: Include account names, numbers, amounts
→ Use neutral language: "Branch has not obtained..." NOT "Branch deliberately..."
→ Report facts, not allegations
→ For large number of similar deficiencies: use annexure with case-wise details

BEFORE FINALISATION:
→ Draft LFAR → Discuss with Branch Head → Get management comments
→ Ensure LFAR observations are consistent with MOC and audit report
→ No contradiction between LFAR, MOC and audit report
→ Review prior year LFAR — have last year's issues been resolved?

CONFIDENTIALITY:
→ LFAR is NOT a public document
→ Share only with: Bank Management, SCA, RBI (if asked)
→ Do not discuss LFAR contents publicly
```

## LFAR vs Audit Report — Key Differences

```
Feature           | Audit Report              | LFAR
Nature            | Opinion document          | Management report
Addressee         | Shareholders/Regulators   | Bank Management / SCA
Public document?  | Yes (filed with ROC/RBI)  | No (confidential)
Format            | SA 700 format             | RBI prescribed questionnaire
Contents          | True & fair view opinion  | Detailed findings on each area
Submission        | By April 30 (approx)      | By June 30
Qualification     | If material finding       | Only LFAR comments
```
